\addcontentsline{toc}{section}{Заключение}
В результате работы методом желаемой ЛАЧХ был спроектирован
последовательный регулятор для системы управления, который обеспечи-
вает необходимые показатели качества на выходе замкнутой системы при
воздействии на неё сигналом с ограниченной скоростью и ускорением.\par
Получены следующие показатели качества:
\begin{itemize}
	\item{$t_p=0.14 c$, при единичном входном воздействии;}
	\item{$\sigma=19\%$, при единичном входном воздействии;}
	\item{$e=0.035$, при линейно возрастающем входном воздействии;}
\end{itemize}\par
Данные характеристики полностью удовлетворяют требуемым.\par
Использование метода желаемой логарифмической амплитудной частотой характеристики позволяет быстро синтезировать регулятор и задать системе необходимые параметры качества\par 
Разработка физической реализации регулятора легко осуществляется путем разбиения передаточной функции на элементарные звенья. Затем каждому звену соответствует четырхполюсник.
