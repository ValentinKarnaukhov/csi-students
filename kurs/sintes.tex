\section{Синтез регулятора}
\subsection{Построение желаемой ЛАЧХ}
Для построения желаемой ЛАЧХ будем использовать номограммы Солодовникова (рисунок 4, 5). Номограммы Солодовникова устанавливают связь между величиной перерегулирования $\sigma$\%, временем переходного процесса tрег, максимальным значением вещественной части АФЧХ Pmax и частотой среза $\omega_{cp}$.\par 
\begin{figure}[h!]
	\begin{center}
		\includegraphics[]{img/solod1}
		\caption{Диаграмма Солодовникова для построения среднечастоной асимптоты}
	\end{center}  
\end{figure}
Исходя из заданного значения перерегулирования $\sigma$, по рисунке 4 можно определить $t_\text{рег}$.
$t_\text{рег}=\frac{3\pi}{\omega n}$
$\omega n=\frac{3\pi}{t_\text{рег}}$
Исходя из соотношения $\omega_{cp}=(0.6...0.9)\omega_n$, найдем частоту среза. $\omega_{cp}=50c^{-1}$. 
Требуемый коэффициент усиления рассчитывается по формуле:
\begin{equation}
K_\text{ж}=\frac{\dot{g}_{max}}{e_{max}}=160
\end{equation}\par 
Низкочастотная асимптота имеет в точке $\omega=1c^{-1}$ ординату равную 20lg(k)=46.6 и наклон -20дБ/дек.\par 
Среднечастотная и низкочастотная асимптоты сопрягаются в том интервале частот, в котором Lж<L$\gamma$. L$\gamma$=20дБ определена по рисунку 5.
\begin{figure}[h!]
	\begin{center}
		\includegraphics[]{img/solod2}
		\caption{график для определения L$\gamma$}
	\end{center}  
\end{figure}\par 
На основании этих данных построим желаемую асимптотическую логарифмическую амплитудно частотную характеристику. График желаемой ЛАЧХ приведен на рисунке 6.
\begin{figure}[h!]
	\begin{center}
		\includegraphics[height=9cm]{img/jlach}
		\caption{Желаемая ЛАЧХ}
	\end{center}  
\end{figure}\par 
По желаемой ЛАЧХ можно определить значения. $\omega_1=4.5c^{-1}$,$\omega_2=17c^{-1}$,$\omega_3=150c^{-1}$.\par
Зная эти значения, можно построить желаемую передаточную функцию:
\begin{equation}
W_\text{ж}=\frac{K_\text{ж}(\frac{1}{\omega_2}s+1)}{(\frac{1}{\omega_1}s+1)(\frac{1}{\omega_3}s+1)s}=\frac{160(\frac{1}{17}s+1)}{(\frac{1}{4.5}s+1)(\frac{1}{150}s+1)s}
\end{equation}
\begin{equation}
W_\text{ж}=\frac{6353s+108000}{s^3+154.5s^2+675s}
\end{equation}

\subsection{Получение передаточной функции регулятора}
Исходя из найденной передаточной функции можно найти передаточную функцию регулятора по формуле:\par
\begin{equation}
W_\text{рег}=\frac{W_\text{ж}}{W_\text{нч}}
\end{equation}
\begin{equation}
W_\text{рег}=\frac{6353s+108000}{s^3+154.5s^2+675s}*\frac{s^2(0.22s+1)}{215}
\end{equation}
При приведении к нормальному виду, получаем функцию регулятора:
\begin{equation}
W_\text{рег}=\frac{1398s^2+30113s+108000}{215s^2+33220s^2+145125}
\end{equation}
